Java Language Features Map [Java 8 - 14]

The Java Language Features Map aims to help you understand the various features of the Java Language across various versions from Java 8 to Java 14 OpenAPI Specification (fka. Swagger Specification) documentation.
It has been created by Rahul Saini .

This tool now shows various Java Language features till the Java 14 version. You can select the appropriate version (Java 8, Java 9, Java 10, Java 11, Java 12, Java 13, or Java 14) by using the links on the top right navigation bar and also use direct links to each version:
  • Java 8
  • Java 9
  • Java 10
  • Java 11
  • Java 12
  • Java 13
  • Java 14

Controls

Mouse scroll up to zoom in
Mouse scroll down to zoom out
Mouse drag to move viewpoint
Move mouse over node to show its documentation
Click on node to expand or collapse it
